# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Project Structure

```
Portfolio/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Stylesheet
├── js/
│   └── script.js      # JavaScript functionality
├── images/            # Project images and photos
└── README.md          # Project documentation
```

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Animated scroll and mobile-friendly hamburger menu
- **Dynamic Typing Effect**: Eye-catching hero section with typing animation
- **Project Showcase**: Grid layout to display your projects
- **Contact Form**: Interactive form for visitors to reach out
- **Scroll Animations**: Elements fade in as you scroll down the page

## Getting Started

1. Open `index.html` in your web browser to view the portfolio
2. Customize the content in `index.html` with your information
3. Update colors and styles in `css/style.css`
4. Add your project images to the `images/` folder
5. Modify `js/script.js` to adjust animations and interactions

## Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --text-color: #1f2937;
    /* ... */
}
```

### Typing Effect
Change the text in `js/script.js`:
```javascript
const textArray = ['Web Developer', 'Designer', 'Problem Solver', 'Creative Thinker'];
```

### Projects
Add or modify project cards in the projects section of `index.html`.
